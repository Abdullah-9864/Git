## The Rule For Every Project

```
Step 1 → Try to build it alone first
Step 2 → Get stuck? Google it
Step 3 → Still stuck? Then watch/read
Step 4 → Finish it
Step 5 → Add one extra feature nobody asked for
Step 6 → Move to the next project
```

> **Don't do all 140. Do 2-3 from each stage then move forward. Quality over quantity.** 🎯
